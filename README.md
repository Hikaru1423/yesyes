# Mansoury - Онлайн книжный магазин

Современный интерфейс книжного магазина с использованием React, TypeScript, Vite и styled-components.

## Технологии

- **Bun** - менеджер пакетов и runtime
- **Vite** - сборщик и dev-сервер
- **React 19** - UI библиотека
- **TypeScript** - типизация
- **styled-components** - стилизация компонентов

## Структура проекта

```
src/
├── components/
│   ├── Header.tsx          # Шапка сайта с навигацией
│   ├── Hero.tsx            # Главный баннер с промо
│   ├── Available.tsx       # Секция "Имееться"
│   ├── NewArrivals.tsx     # Секция "Новинки" с каруселью
│   ├── BooksSection.tsx    # Секция с любимыми книгами
│   ├── DiscountCard.tsx    # Секция с картой скидок
│   └── Footer.tsx          # Подвал сайта
├── App.tsx                 # Главный компонент
├── main.tsx                # Точка входа
└── index.css               # Глобальные стили
```

## Установка и запуск

```bash
# Установка зависимостей
bun install

# Запуск dev-сервера на порту 5174
bun run dev

# Сборка для продакшена
bun run build

# Превью продакшен сборки
bun run preview
```

## Разработка

Проект настроен на работу на порту **5174** (см. `vite.config.ts`).

Все компоненты используют styled-components для стилизации и полностью типизированы TypeScript.

## Особенности

- Адаптивный дизайн
- Компонентный подход
- Типизированные компоненты
- Стилизация через styled-components
- Без использования UI библиотек
